## 🛠 Technical Overview: SmartStaccer Contract

The backend of Staccer utilizes a two-phase withdrawal logic to ensure maximum security for both the platform and the creator.

1. **NewSale**: Triggered when a fan buys content. Funds are locked for 21 days.
2. **ClaimSale**: After the lockup, funds move to the creator's "Available Balance."
3. **Withdraw**: Creators can withdraw their liquid balance to their wallet at any time.



### Built With
